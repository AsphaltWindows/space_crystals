## Content

Updated `artifacts/designer/design/camera.md` to add a **Starting Position** section. At game start, the camera must center on the local player's primary structure — Deployment Center for GDO, starting Tunnel for Syndicate.

## QA Instructions

1. Start a new game as GDO. Verify the camera is centered on the Deployment Center when the game begins.
2. Start a new game as Syndicate. Verify the camera is centered on the starting Tunnel when the game begins.
3. Confirm no manual panning is needed to see your starting structure — it should be visible and centered in the viewport from the first frame.
