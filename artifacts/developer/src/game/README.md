# game/

Core game logic modules for Space Crystals RTS.

## Structure

- **types.rs** — Faction definitions, faction resource types (GDO, Syndicate, Cults, Colonists), PlayerResources
- **utils.rs** — Game-level shared utilities
- **combat/** — Attack systems, turrets, and projectiles
- **units/** — Unit spawning, movement, pathfinding, and commands
- **world/** — Map/tile system, resource nodes, faction setup, and selection
